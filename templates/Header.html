Home automation demo

-This is a main page example of the website i developed to control devices in the home
-Hover over items to see what they do.
-This page is made with HTML (obviously), css, JavaScript (inc jquery and AJAX) and also bootstrap material design hosted on a bottle python backend webserver which is run on a raspberry Pi
-All the actions are carried out through asyncronous AJAX requests triggering the desired action in the python backend< br> -The entire webpage is dynamic so adjusts to fit (almost) any size screen
-Use the navbar to navigate between pages:

This is the control page, here you can contorl the virtual home.
The left column is for controlling heating status and percentage.
Each button corresponds to a particular temperature percentage for quick use while the 'MORE' button allows you to get a more specific value.
The right column is devoted to controlling relays which are connected to various devices such as a PC and LED lights
pressing a button causes a request to be sent in the background to the host computer, telling it to do something
then it sends the new status of the device in return and the site updates the status of the cirles to represent the change

This page contains a shopping list which can be easily edited, allowing the list to be accessed anywhere with internet access
The list is stored on a mySQL server on the webserver machine.
To edit, just click the pencil symbol over the box you wish to chane or hit the plus symbol for a new entry.
